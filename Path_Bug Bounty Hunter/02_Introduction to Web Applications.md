## Front End Components

### HTML  
  
Q: What is the HTML tag used to show an image?  
A: `<img>`

### Cascading Style Sheets (CSS)
  
Q: What is the CSS "property: value" used to make an HTML element's text aligned to the left?  
A: `text-align: left;`
